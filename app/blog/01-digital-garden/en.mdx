# 01 · Welcome to My Digital Garden

> Somewhere between logic and emotion, finding my place slowly.

This is the first post in my digital garden.

I’m a Computer Science student at **Beijing Technology and Business University**, and my MBTI type is ISTJ.

I tend to break problems down, analyze and optimize them — understanding the world the same way I write code. But gradually I realized that if only “rationality” and “efficiency” remain, a person can become a bit one-dimensional.

So I want to keep a space to document not only technology, but also language, emotions, and growth.

---

## 🛠️ Tech · Systems, Code, and Tinkering

For a CS student, understanding the underlying system feels almost like an obsession.

Recently I’ve been working through operating system labs. From environment setup to kernel compilation errors, I debug line by line in GitHub Codespaces. Honestly, the process is frustrating, but when the system finally runs, that feeling of “the logic holds together” is incredibly satisfying.

This website itself is also an exercise:

```text
Next.js 14
Supabase
```

Fixing Turbopack errors, handling cache revalidation, dealing with deployment issues — all very specific problems, but solving them makes me feel like I’m genuinely improving, not just “able to code.”

I want this place to be a long-term repository for my technical notes.

## 🗺️ What’s Next

This garden won’t grow into a forest overnight. It will evolve slowly.

Besides technical notes, I also plan to share photography and baking records. Life can’t be only about compiling and debugging.
